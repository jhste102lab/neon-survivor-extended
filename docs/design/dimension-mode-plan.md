# 차원 모드 개편 기획서

- 상태: 구현 전 확정 기획
- 작성일: 2026-07-02
- Parent epic: [#33](https://github.com/jhste102lab/neon-survivor-extended/issues/33)
- 목적: 10분 이후 반복 작업감을 줄이고, 후반을 피지컬/판단/처치/보상 선택이 섞인 차원 정복 구조로 전환한다.

## 1. 핵심 문제와 목표

### 유저 피드백 요약

- 15분 전후부터 기존 뱀서류/핵앤슬래시 반복감이 강해져 지루하다.
- 15분 보스 이후에는 잡몹 물량보다 빠른 템포, 다양한 투사체, 패턴 판단, 멈춰야 하는 순간이 필요하다.
- 스토리/완결 후 무한 모드처럼 구조적 끝맺음이 있으면 좋다.
- 이벤트 구역은 작고, 차라리 다른 차원에서 다른 적과 싸우는 편이 낫다.
- 플레이어 성장이 지수적으로 커지는 만큼 적/보스 성장과 후반 밸런스가 필요하다.
- 스킬/패시브 설명과 실제 수치가 다른 부분이 있고, 적/보스 도감에 체력/공격력 등 구체 수치가 부족하다.

### 재미 우선 목표

1. 10분까지는 기존 성장/처치 재미를 유지한다.
2. 10분 이후에는 후반 목적을 `차원 정복`으로 전환한다.
3. 잡몹 물량만 늘리는 방식이 아니라, 차원별로 다른 룰/목표/비주얼/보상을 제공한다.
4. 플레이어가 키운 빌드는 차원에서도 유지한다.
5. 막히면 외부 전장에서 파밍하고 다시 도전할 수 있게 한다.
6. 끝없는 반복감은 줄이되, 완전 정복 이후 자유 플레이는 허용한다.

## 2. 전체 게임 흐름

```text
0~10분: 기존 생존/성장/처치 루프
10분: 신규 관문 보스 `균열 문지기` 등장
균열 문지기 처치: 외부 전장에 허브 진입 포털 생성
허브 진입 포털: 5초 머물면 차원 선택 허브로 이동
허브: 8개 차원 포탈을 처음부터 표시, 자유 선택
차원: 목표 달성 시 정복, 체력 0이면 붕괴 탈출
외부 전장: 언제든 나가서 파밍/회복/보스 처치 가능
8개 차원 모두 정복: 성공 판정/기록/연출 후 버튼 없이 자유 플레이 지속
완전 정복 후: 모든 차원 재입장 가능, 최초 보상 재지급 없음
```

## 3. 확정된 상위 결정

- 기본 런은 10분 관문 보스 이후 차원 허브가 열리는 구조로 전환한다.
- 중간 성공 종료 포탈은 만들지 않는다.
- 8개 차원을 전부 구현하고, 한 판에서 전부 클리어 가능하게 한다.
- 8개 전부 클리어 시 성공 판정은 하지만 게임을 끊지 않는다.
- 허브는 자유롭게 들어가고 나올 수 있다.
- 허브/차원 안에서는 외부 전장 시간이 정지하고, 상태는 리셋 없이 보존된다.
- 차원 안에서도 0.5x/1x/2x/3x 배속을 모두 허용하며 페널티는 없다.
- 차원 안에서도 자동무기, 동료, 진화, 오버드라이브는 유지한다.
- PC 우선 설계다. 모바일은 깨지지 않는 정도만 고려한다.
- 구현 전 Phase 0으로 수치 표기/도감 신뢰도 문제를 먼저 정리한다.

## 4. 10분 관문 보스: 균열 문지기

관련 이슈: [#37](https://github.com/jhste102lab/neon-survivor-extended/issues/37)

### 역할

- 본편 10분 구간을 마무리하고 차원 허브를 여는 관문 보스.
- 기존 `군집 코어`는 후반 외부 전장/엔드리스 보스로 남긴다.
- 목적은 차원 모드 진입이며, 너무 어려워서 콘텐츠 접근을 막으면 안 된다.

### 난이도 목표

- 기존 유저 통과율: 75~85%
- 일반 유저 통과율: 60~75%
- 초심자 통과율: 40~55%
- 전투 시간 목표: 60~90초

### 패턴 방향

- 원형 탄막 + 안전 틈
- 예고 레이저
- 느린 추적 장판
- 체력 30% 이하에서 패턴 간격 단축
- 잡몹 소환은 없거나 아주 적게

### 처치 보상

- 허브 진입 포털 생성
- 외부 전장 일부 정리/시야 확보
- 포털 주변 안전지대 생성

## 5. 허브와 포탈 구조

관련 이슈: [#38](https://github.com/jhste102lab/neon-survivor-extended/issues/38), [#39](https://github.com/jhste102lab/neon-survivor-extended/issues/39)

### 포탈 종류

| 포탈 | 역할 |
|---|---|
| 허브 진입 포털 | 외부 전장 ↔ 차원 허브 왕복. 10분 보스 처치 후 생성. |
| 차원 포탈 | 허브 → 각 차원 진입. 8개 모두 처음부터 표시. |
| 귀환 포탈 | 허브 → 외부 전장 복귀. 자유 출입. |

중간 성공 종료용 탈출 포탈은 만들지 않는다.

### 허브 규칙

- 허브에는 적이 없다.
- 허브에서는 외부 전장 시간이 정지한다.
- 외부 전장의 적, 보스 타이머, 웨이브 타이머, 드롭, 위험도는 보존된다.
- 8개 차원 포탈은 처음부터 전부 보인다.
- 미정복 차원은 진입 가능하다.
- 완전 정복 전에는 이미 정복한 차원 재입장을 막고 `정복 완료` 상태로 표시한다.
- 8개 전부 정복 후에는 모든 차원 재입장이 열린다.

### 포탈 설명 UI

허브 포탈은 위키가 아니라 선택 공간이다. 각 포탈은 핵심 정보만 표시한다.

```text
탄막 성운
위험도 ★★★
목표: 성운 코어 파괴
보상: 성운 감각
룰: 탄막 사이를 읽고 회피
```

표시 항목:

- 차원 이름
- 위험도
- 목표
- 고정 유물
- 한 줄 룰 설명
- 정복 여부

## 6. 외부 전장: 자유 파밍과 성장 연동

관련 이슈: [#40](https://github.com/jhste102lab/neon-survivor-extended/issues/40)

### 역할

- 차원이 어렵거나 체력이 낮을 때 다시 나가서 파밍/회복/빌드 보강을 할 수 있는 공간.
- 파밍은 허용한다.
- 단, 허브 왕복으로 리세마라가 되면 안 된다.

### 규칙

- 10분 이후에도 외부 전장은 계속 진행된다.
- 허브/차원에 들어가면 외부 전장 시간만 정지한다.
- 돌아오면 정지했던 상태 그대로 이어진다.
- 보스도 등장한다.
- 외부 보스는 파밍 루프의 핵심 이벤트다.
- 보스 처치 보상은 자동 지급이며 기존 성장/회복/드롭/점수 중심이다.
- 외부 보스는 차원 고유 유물이나 차원 추가 보너스를 주지 않는다.

### 외부 보스 보상

- 큰 XP 보석 또는 레벨업 상당 보상
- 회복 드롭 1~2개
- 폭탄/자석 중 일부
- 점수 보너스
- 낮은 확률의 보물상자

### 적/보스 성장

- 플레이어가 성장하는 만큼 적/보스도 소프트 스케일링한다.
- 권장 감각: 시간 압박 + 플레이어 파워의 60~75%를 반영.
- 100% 따라오면 성장 보람이 사라지고, 0%면 파밍으로 차원이 무의미해진다.

## 7. 차원 공통 규칙

관련 이슈: [#39](https://github.com/jhste102lab/neon-survivor-extended/issues/39)

### 입장 스케일링

차원 입장 시 현재 플레이어 상태를 스냅샷한다.

스냅샷 후보:

- 플레이어 레벨
- 무기 개수와 무기 총 레벨
- 진화 무기 개수
- 패시브 총 레벨
- `stat().dmg`, `stat().cd`, `stat().spd`
- 동료 수/역할
- 현재 체력/최대 체력
- 이미 정복한 차원 수
- 외부 전장 위험도

스케일링 원칙:

- 적 HP는 약하게만 따라온다.
- 주요 난이도는 패턴 밀도, 목표 압박, 예고 시간, 안전지대 크기, 장치 수로 올린다.
- 플레이어가 강해진 보람은 남긴다.

권장 감각:

```text
roomDifficulty = 1 + 정복한 차원 수 * 0.16
enemyHp *= 1 + (roomDifficulty - 1) * 0.35
patternPressure *= roomDifficulty
```

정확한 공식은 밸런스 테스트로 조정한다.

### 시간과 목표

- 사용자에게 시간 제한을 표시하지 않는다.
- 시간 초과 실패나 시간 페널티는 없다.
- 내부 밸런스 목표만 둔다.
- 1~3배속과 0.5x를 고려해 차원 하나는 게임 시간 기준 보통 120~150초, 길어도 180초 전후 자연 완료를 목표로 한다.
- UI는 타이머 대신 목표 진행도 중심이다.

예:

- `성운 코어 안정화 62%`
- `발전기 2/4 파괴`
- `중력 앵커 1/3`
- `심판자 체력 48%`
- `계약 라운드 2/3`

### 실패

- 실패 조건은 플레이어 체력 0으로 통일한다.
- 체력 0 시 즉시 게임오버가 아니라 차원 붕괴 연출 후 자동 탈출한다.
- 직접 탈출 균열로 이동하는 인터랙션은 넣지 않는다.
- 복귀 체력은 25~35% 범위가 적절하다.
- 해당 차원은 미정복 상태로 유지된다.
- 재도전 가능하다.

### 성공

- 목표 달성 시 1~1.5초 정도 짧은 안정화/정화 연출을 보여준다.
- 고정 유물을 지급한다.
- 추가 보너스 3장 중 1개를 선택한다.
- 허브로 복귀하고 포탈은 `정복 완료` 상태가 된다.

### 차원 안 전투 시스템

- 자동무기 유지.
- 동료/진화/오버드라이브 유지.
- 수동 공격 추가 없음.
- 목표물/보스/장치 우선 타겟팅 필요.
- 위험 탄막/레이저/장판 가독성을 위해 아군 파티클, 피해 숫자, 과도한 이펙트는 줄인다.

### 차원 안 드롭과 안전망

XP/레벨업 폭주는 제한하되 생존 드롭은 유지/강화한다.

- 입장 시 체력 60% 미만이면 60%까지 보정 회복.
- 체력 60% 이상이면 소형 보호막 지급 가능.
- 체력 35% 미만이고 최근 회복이 없으면 회복 드롭 예약.
- 체력 20% 미만이면 긴급 보호막/회복 드롭.
- 목표 일부 달성 시 소형 회복/보호막 드롭.
- 방 클리어 후 다음 방을 위해 체력 50~70%까지 보정 가능.

## 8. 차원 보상

### 고정 유물

각 차원을 처음 클리어하면 차원별 고정 유물을 자동 획득한다.

| 차원 | 유물 | 효과 방향 |
|---|---|---|
| 탄막 성운 | 성운 감각 | 피격 직전 짧은 슬로우, 쿨다운 있음 |
| 기계 감옥 | 절연 장갑 | 레이저/장판 피해 1회 감소 또는 무효 |
| 중력 우물 | 중력 파열 | 주기적으로 적을 끌어모은 뒤 작은 폭발 |
| 심판자의 결투 | 심판의 표식 | 보스/엘리트 피해 증가 |
| 역병 정원 | 정화 포자 | 독/오염 정화와 소형 회복 |
| 거울 회랑 | 반사막 | 적 탄환 1개 반사 또는 무효 |
| 열차 위 전장 | 추진 코어 | 짧은 이동 가속/속도 버스트 |
| 도박장의 균열 | 행운 계약 | 보상 선택지/드롭 품질 등 선택·운 요소 강화 |

### 추가 보너스 3장

차원 클리어 후 고정 유물 외에 3장 중 1개를 선택한다.

구성:

- 생존 보너스 1장
- 전투 보너스 1장
- 경제/위험 보너스 1장

예:

```text
[재정비] 체력 35% 회복
[과충전] 가장 높은 레벨 무기 피해 +8%
[위험 계약] 다음 차원 위험도 +1, 클리어 보상 +1
```

주의:

- 너무 큰 단순 피해 보너스는 금지.
- 차원 고유 유물보다 특별해지면 안 된다.
- 설명은 실제 수치를 포함해 짧게 쓴다.

## 9. 8개 차원 상세

관련 이슈: [#41](https://github.com/jhste102lab/neon-survivor-extended/issues/41) ~ [#48](https://github.com/jhste102lab/neon-survivor-extended/issues/48)

### 9.1 탄막 성운

- 역할: 순수 탄막 회피 + 코어 파괴.
- 목표 UI: `성운 코어 안정화 0~100%`
- 클리어: 성운 코어 파괴 또는 안정화 100%.
- 적/장치: 중앙/상단 성운 코어, 탄막 노드 3~5개, 일반 잡몹 없음 또는 극소수.
- 패턴: 원형 탄막+틈, 나선 탄막, 느린 대형 탄, 화면 가장자리 별조각 탄.
- 타겟팅: 코어/탄막 노드 우선.
- 유물: 성운 감각.
- 드롭: 보호막/회복 중심, 폭탄 제한.
- 에셋: 우주, 성운, 스페이스 슈터, 탄막 스프라이트.
- 성공 연출: 성운 코어 안정화, 별구름 정렬.
- 실패 연출: 성운이 찢어지고 별조각이 빨려 들어감.

### 9.2 기계 감옥

- 역할: 레이저/문/안전칸 판단 + 장치 파괴.
- 목표 UI: `발전기 0/4 파괴`
- 클리어: 발전기 4개 파괴.
- 적/장치: 발전기 4개, 레이저 포탑, 닫히는 문/전기벽, 소형 감시 드론.
- 패턴: 예고선 후 레이저, 칸 단위 전기 바닥, 문 폐쇄, 예측 사격.
- 타겟팅: 발전기/포탑 우선.
- 유물: 절연 장갑.
- 드롭: 발전기 파괴 시 회복/보호막.
- 에셋: 메탈 타일, 레이저 게이트, 전기 장치.
- 성공 연출: 감옥 전원 차단, 문 개방.
- 실패 연출: 레이저 과부하, 감옥 붕괴.

### 9.3 중력 우물

- 역할: 중력장 안에서 앵커 파괴 + 웨이브 처치.
- 목표 UI: `중력 앵커 0/3 파괴`
- 클리어: 중력 앵커 3개 파괴 후 중앙 우물 안정화.
- 적/장치: 중앙 중력 우물, 앵커 3개, 중력에 끌리는 웨이브 적, 느린 대형 파편 탄.
- 패턴: 중앙 끌림, 주기적 바깥 밀림, 앵커 주변 회전 탄막, 앵커 파괴 시 중력 변화.
- 타겟팅: 앵커 우선.
- 유물: 중력 파열.
- 드롭: 웨이브 처치 드롭, 회복/폭탄 적당히 허용, XP 과다 제한.
- 에셋: 블랙홀, 우주 잔해, 왜곡 파티클, 중력 렌즈.
- 성공 연출: 우물 수축과 파편 정렬.
- 실패 연출: 중력 폭주로 화면 중심 흡입.

### 9.4 심판자의 결투

- 역할: 느린 패턴 보스 결투.
- 목표 UI: `심판자 체력` 또는 처치 진행도.
- 클리어: 심판자 처치.
- 적/장치: 잡몹 거의 없음, 느린 패턴 보스 중심.
- 패턴: 십자 레이저, 부채꼴 탄막, 안전지대 장판, 움직이는 패턴과 멈춰야 하는 패턴 교대.
- 타겟팅: 심판자 우선.
- 유물: 심판의 표식.
- 드롭: 보스 페이즈 전환 시 회복/보호막 보정.
- 에셋: 신전/법정/거대한 문양/보스 아레나.
- 성공 연출: 심판 문양 완성.
- 실패 연출: 판결장 붕괴.

### 9.5 역병 정원

- 역할: 독성 영역 관리 + 둥지/코어 파괴.
- 목표 UI: `독성 둥지 0/N 정화`
- 클리어: 독성 둥지 파괴 후 포자 코어 처치.
- 적/장치: 독성 둥지, 포자 장판, 생체 덩굴, 독성 적.
- 패턴: 오염 확산, 포자 장판, 독성 식물 발사체.
- 유물: 정화 포자.
- 드롭: 정화/파괴 시 소형 회복, 독 피해 안전망.
- 에셋: 독성 식물, 포자, 생체 덩굴, alien plant.
- 성공 연출: 정원 정화.
- 실패 연출: 포자 폭발.

### 9.6 거울 회랑

- 역할: 진짜/가짜 판단 + 반사 탄막.
- 목표 UI: `진짜 거울핵 0/3 파괴`
- 클리어: 진짜 거울핵 3개 파괴.
- 적/장치: 진짜 핵, 가짜 핵, 프리즘 장치.
- 패턴: 반사탄, 가짜 핵 반격, 프리즘 레이저.
- 타겟팅: 가짜 핵 때문에 불쾌하지 않도록 전용 타겟 규칙 필요.
- 유물: 반사막.
- 드롭: 핵 파괴 시 보호막/회복.
- 에셋: 유리, 프리즘, 크리스탈, 빛줄기.
- 성공 연출: 거울 회랑 안정화.
- 실패 연출: 화면이 조각나며 붕괴.

### 9.7 열차 위 전장

- 역할: 움직이는 전장 + 침입 적 처치.
- 목표 UI: `강습 웨이브 0/N`
- 클리어: 강습 웨이브 방어 및 마지막 강습선 격퇴.
- 적/장치: 드론 습격, 장애물, 강습선, 움직이는 칸/컨베이어.
- 패턴: 횡스크롤식 압박, 장애물 통과, 다방향 침입.
- 유물: 추진 코어.
- 드롭: 웨이브 방어 중 회복/폭탄 허용.
- 에셋: 열차, 수송선, 컨베이어, sci-fi vehicle.
- 성공 연출: 열차/수송선 방어 완료.
- 실패 연출: 선로/수송선 붕괴.

### 9.8 도박장의 균열

- 역할: 리스크/보상 선택 전투.
- 목표 UI: `계약 라운드 1/3`
- 클리어: 계약 라운드 3회 승리.
- 적/장치: 카드/룰렛/슬롯 테마 적과 라운드별 계약 효과.
- 패턴: 라운드마다 3개 계약 카드 제시, 선택 후 전투.
- 랜덤성: 제어된 랜덤. 결과/위험/보상은 선택 전 명확히 표시.
- 금지: 숨겨진 페널티, 설명과 다른 효과, 회피 불가능 조합.
- 유물: 행운 계약.
- 에셋: 카지노, 룰렛, 카드, 슬롯, 네온 칩.
- 성공 연출: 잭팟 안정화.
- 실패 연출: 슬롯/카드/룰렛 붕괴.

## 13. Phase 0: 수치 표기와 도감 신뢰도

관련 이슈: [#35](https://github.com/jhste102lab/neon-survivor-extended/issues/35), 기존 관련 이슈 [#32](https://github.com/jhste102lab/neon-survivor-extended/issues/32)

차원 모드 전에 처리해야 할 품질 이슈다.

### 문제

- 오버클럭/무기 가속 등 설명이 실제 수치와 다르게 읽힐 수 있다.
- 현재 실제 쿨다운 배율은 `Math.pow(0.93, hasteLevel)`이므로, 표현은 `무기 공격 간격 -7%`처럼 실제 계산과 맞아야 한다.
- 적/보스 도감에 체력, 공격력, 속도, 주요 특성 같은 구체 수치가 부족하다.
- 방어력/관통저항처럼 실제 시스템에 없는 수치를 허위로 표시하면 안 된다.

### 기준

- 무기/패시브/초월/진화 설명은 실제 수치 기반으로 정정한다.
- 도감에는 실제 데이터에서 가져온 HP/공격력/속도/특성/등장 시점을 표시한다.
- 없는 시스템은 `해당 없음` 또는 미표시로 처리한다.
- 차원 유물/보상 설명도 짧지만 실제 효과 수치를 포함한다.

## 16. 검증 기준

관련 이슈: [#50](https://github.com/jhste102lab/neon-survivor-extended/issues/50)

### 기능 검증

- 10분 시점 균열 문지기가 등장하고 처치 후 허브 진입 포털이 열린다.
- 허브 왕복 시 외부 전장이 리셋되지 않는다.
- 8개 차원을 원하는 순서로 진입할 수 있다.
- 차원 클리어/실패/재도전/완전 정복 흐름이 작동한다.
- 완전 정복 후 모든 차원 재입장이 열린다.
- 재입장은 최초 보상을 재지급하지 않는다.

### 재미/밸런스 검증

- 관문 보스가 너무 어렵거나 싱겁지 않다.
- 각 차원이 서로 다른 플레이 이유를 가진다.
- 차원이 색만 다른 같은 방처럼 느껴지지 않는다.
- 외부 파밍이 허용되지만 차원을 무의미하게 만들지 않는다.
- 스케일링이 100% 따라와 성장 보람을 없애지 않는다.
- 0.5x 사용이 허용되지만 보상 페널티는 없다.
- 3x에서도 차원이 너무 허무하게 끝나지 않는다.

### 기술 검증

- `npm run check:syntax`
- `npm run verify`
- 가능하면 브라우저 스모크 테스트
- 변경 코드가 있을 경우 `jhste-skills guard --scope changed --format text --fail-on error`
- 비트리비얼 코드 변경은 red-team review 수행

## 17. 주요 리스크와 방어책

| 리스크 | 방어책 |
|---|---|
| 8개 차원이 색만 다른 같은 방이 됨 | 각 차원에 다른 목표/룰/유물/연출을 강제 |
| 후반이 다시 너무 길고 지루함 | 성공 판정은 8개 정복 시 명확히 주고, 이후는 자유 플레이로 처리 |
| 외부 파밍으로 차원 무의미화 | 입장 스냅샷과 60~75% 성장 연동 스케일링 |
| 적도 너무 따라와 성장 보람 상실 | HP 스케일은 약하게, 패턴 밀도 중심 조정 |
| 자동무기 타겟이 장치를 못 때림 | 차원 목표물 우선 타겟팅 필수 |
| 탄막/이펙트 가독성 악화 | 차원 안 아군 이펙트/피해 숫자 제한 |
| 에셋 라이선스 문제 | Noto Emoji SVG 등 공개 에셋을 로컬 포함하고 `assets/dimensions/NOTICE.md`에 출처/라이선스 기록 |
| 수치 설명 불신 | Phase 0에서 실제 수치 기반 설명/도감 정비 |

## 18. 구현 메모

- 포털 장식은 공개 오픈소스 Noto Emoji SVG를 로컬 에셋으로 포함한다. 세부 파일과 라이선스 출처는 `assets/dimensions/NOTICE.md`에 기록한다.
- 10분 시점에는 기존 보스가 남아 있어도 균열 문지기가 우선권을 가진다. 기존 보스는 균열 연출로 흡수되어 차원 허브 진입이 오래 막히지 않는다.
- 브라우저 검증에서는 8개 차원의 직접 진입/클리어/유물 지급/정복 플래그를 강제 시나리오로 확인한다.
