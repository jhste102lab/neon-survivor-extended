# v1.2.0 릴리즈 노트

## 차원 모드 오버홀

- 10분 시점에 기존 후반 루프를 바로 반복하지 않고, 먼저 **균열 문지기** 관문 보스가 등장합니다.
- 균열 문지기를 처치하면 차원 허브 포털이 열리며, 포털 안에 5초 머물러 허브로 진입합니다.
- 허브에서는 8개 차원을 원하는 순서로 고를 수 있고, 외부 전장으로 자유롭게 복귀해 파밍한 뒤 다시 도전할 수 있습니다.
- 차원 안에서 체력이 0이 되면 즉시 게임오버가 아니라 붕괴 연출 후 허브로 자동 탈출합니다.

## 8개 신규 차원

- **탄막 성운**: 탄막을 읽고 성운 코어를 파괴합니다.
- **기계 감옥**: 레이저와 전기벽 사이에서 발전기를 파괴합니다.
- **중력 우물**: 끌림/밀림을 이용해 웨이브와 중력 앵커를 정리합니다.
- **심판자의 결투**: 느린 고위험 패턴 보스를 처치합니다.
- **역병 정원**: 독성 영역을 관리하며 둥지를 정화합니다.
- **거울 회랑**: 진짜/가짜 핵을 구분하고 반사탄을 피합니다.
- **열차 위 전장**: 움직이는 전장에서 강습 웨이브를 방어합니다.
- **도박장의 균열**: 공개된 위험/보상 계약을 고르고 라운드를 승리합니다.

## 보상 / 밸런스

- 각 차원은 고정 유물을 지급하고, 별도의 차원 보상 카드 3장 중 하나를 선택하게 합니다.
- 체력이 낮을 때 안전 회복 드롭과 보호막 보정이 들어가 허탈사를 줄입니다.
- 플레이어 빌드 성장, 외부 파밍, 차원 클리어 수에 따라 적 체력과 패턴 밀도가 완만하게 상승합니다.
- 외부 전장 보스 처치에 붙이려던 별도 추가 차원 보상은 제거하고, 기존 보스 보상과 차원 내부 보상을 분리했습니다.

## UI / 문서 / 에셋

- 차원 HUD, 허브 포털 렌더링, 차원 배경, 포털 충전 표시를 추가했습니다.
- Google Noto Emoji SVG를 차원 포털 장식 에셋으로 포함하고 출처/라이선스를 `assets/dimensions/NOTICE.md`에 기록했습니다.
- 도감에 차원 탭, 균열 문지기, 실제 수치 행을 추가했습니다.
- 오버클럭/무기 가속 패시브 설명을 실제 효과인 `무기 공격 간격 -7%` 기준으로 바로잡았습니다.
- 상세 기획/이슈 대응 문서는 `docs/design/dimension-mode-plan.md`에 정리했습니다.
