# v1.1.6 Release Notes

## Highlights

- Tuned the first 10:00 endless Cluster Core so it is smaller, slower, easier to read, and gives the player more room to escape before later 13:00, 16:00, 19:00 fusion bosses continue the 3-minute loop.
- Replaced the visible dense-swarm mass-layer path with cohort-style normal-enemy compaction and cached movement, so crowded enemies stay smooth without the low-frame layer feel.
- Restricted compaction away from the player and visible screen to avoid nearby monsters disappearing or jumping position during play.
- Added a rainbow Neon outline only to companion Neon upgrade cards, keeping normal weapon cards and card text readable.

## Patch notes

- 10분 첫 엔드리스 합체 보스가 더 작고 느리게 등장해서 도망칠 여지가 생김.
- 많이 쌓인 일반몹은 mass layer 대신 코호트 압축으로 처리해서 저프레임처럼 보이는 느낌을 줄임.
- 화면 안/플레이어 주변 몹은 압축하지 않아서 다가오다 사라지거나 위치가 바뀌는 체감을 완화함.
- 동료 네온 카드만 무지개 테두리로 강조하고, 일반 네온 무기 카드는 기존 카드처럼 유지함.
