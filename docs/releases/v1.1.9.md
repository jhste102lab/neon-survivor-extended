# v1.1.9

## 엔드리스 보스 포식 리워크

- 엔드리스/합체 보스의 포식 능력을 3초 예고 후 발동하도록 조정했습니다.
- 포식 예고 중 보스가 노리는 드랍/보석은 쟁탈 상태가 되어 TAB 집중 신규 보관, E 회수, 자석/냉기장 원격 흡수가 막히고 직접 접근으로만 선점할 수 있습니다.
- 포식 발동 중 플레이어가 대상 픽업에 닿으면 즉시 획득 대신 흡수 차단이 발생하며, 픽업은 튕겨 나가 짧게 보호됩니다.
- 상자는 보스 포식 대상에서 제외했습니다.
- 보스가 닭과 경험치 보석을 먹으면 체력을 회복하도록 다시 열되, 포식 1회/분당 회복 상한과 저체력 회복 감쇠를 적용했습니다.
- 보스가 폭탄을 먹으면 이번 포식 사이클에서 회복한 체력의 10%를 기준으로 역류 피해를 받으며, 포식 1회 총 피해 상한을 적용했습니다.
- 보스가 자석을 먹으면 현재 XP 진행도가 줄고, XP가 부족하면 최근 강화/획득 무기 1개가 짧게 과열/봉인될 수 있습니다. 레벨다운과 영구 무기 삭제는 발생하지 않습니다.

## 무기 봉인 / 후반 성장

- 무기 침묵을 다중 무기 봉인으로 확장했습니다. 합체/메가 보스는 최대 6개, 일반 엔드리스 보스는 시간대에 따라 3~5개를 봉인합니다.
- 봉인 대상은 최소 활성 무기 수와 상위 무기 동시 봉인 제한을 지켜 빌드 전체가 꺼지는 상황을 줄였습니다.
- 봉인은 보스 처치 시 해제되며, 시간 경과와 순차 해제 보험을 둡니다.
- 10분 이후 일반 몬스터 경험치 일부는 즉시 지급하고, 오래 방치된 보석은 일부 가치로 자동 정산해 후반 보석 clutter를 줄였습니다.
- 직접 획득/자석 획득은 자동 정산보다 높은 가치를 유지합니다.

## UI / 피드백

- 보스가 노리는 보석/드랍에 링 표시를 추가했습니다.
- 봉인된 무기 슬롯에 잠금 배지와 남은 시간을 표시합니다.
- 포식 회복, 폭탄 역류, 자석 XP 손실, 무기 복구 문구를 4개 지원 언어에 추가했습니다.

## 검증

- `npm run verify` 통과
- `node scripts/headless-runtime-sim.mjs` 기능 스모크 통과: PC/모바일 20분 가속 도달, 0.5x 이동 정상, 보스 포식/봉인 상호작용 정상
- 헤드리스 렌더링 성능 경고는 기능 실패와 분리해 출력하도록 정리했습니다. 현재 환경에서는 10분 전 고밀도 렌더 프레임에서도 경고가 발생하므로 별도 성능 튜닝 대상으로 봅니다.
