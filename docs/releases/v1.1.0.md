# v1.1.0 Release Notes Draft

## Highlights

- Expanded weapon evolution coverage from 8 to 24 weapons, so every weapon in the current arsenal now has an evolution path.
- Added 16 new non-healing, non-shielding evolution designs focused on damage identity, crowd control, targeting pressure, and movement-driven play.
- Updated late-game evolution content, localized evolution copy, and README feature documentation for the full 24-evolution arsenal.
- Added the recurring 10-minute Cluster Core fusion boss loop, with later fusion bosses scaling upward and endless bosses gaining randomized extra patterns.
- Added in-run 1x/2x/3x speed controls, a rainbow animated XP bar, weapon-count text on level-up, shorter drop lifetimes, and clearer title-screen nickname guidance.
- Retuned late pressure so enemy speed and density begin ramping from 6 minutes and become more noticeable around 7–8 minutes, while preserving high late-game enemy counts.
- Added mass-enemy render batching, trimmed enemy glow sprite bounds, and layer reuse so large 15–20 minute swarms remain visually crowded with fewer frame hitches.
- Removed magnet and idle-pressure banner spam; idle pressure now suppresses passive sustain, reduces chicken/shield recovery, and lets idle missiles pierce barrier/invulnerability.
- Strengthened the 10-minute Cluster Core with higher contact damage, denser/faster ring bullets, stronger hazard patterns, faster summons, and partial barrier-piercing boss pressure.
- Reduced push/pull control strength and added late-game slow resistance so enemies keep approaching even under heavy crowd-control builds.
- Bumped the local build version to v2.1 phase2 and advanced the leaderboard ruleset to `phase2-2026-06-20-evolution24`.

## New evolution paths

- Quantum Lance, Gravity Barrage, Breaching Shotgun, Hive Drone, Singularity Well, Corrosion Blade, Starfall Volley, Sprint Minefield, Jackpot Disc, Chrono Prison, Overload Railgun, Bio-Corrosion Mist, Overheat Phoenix, Resonance Shatter, Absolute-Zero Spear, and Accelerated Satellite Grid.
