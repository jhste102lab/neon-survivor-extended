# v1.1.3 Release Notes

## Highlights

- Added an adaptive late-game clarity system that reduces non-critical visual noise while preserving enemies, hazards, enemy bullets, and player-state readability.
- Added a locked `0.5x` precision speed control that unlocks after 08:00 and slows the full game clock, simulation, cooldowns, and drop timers together.
- Added a visible invulnerability ring around the player so post-hit safety windows are easier to read during crowded fights.
- Added stack merging for nearby chicken, bomb, and magnet drops, with capped 1.3x stack scaling for healing and bomb damage while keeping chests separate.
- Reworked magnet pickup handling to keep a limited gem pull-in animation and immediately bank excess gems in high-load situations.
- Replaced the title-screen Cloudflare build link with a GitHub repository link.
- Enabled localhost builds to use the public Cloudflare leaderboard API by default, while keeping `?remoteLb=0` as a local-only opt-out.
- Improved local/global leaderboard fallback messaging when the remote leaderboard cannot be reached.
