# v1.1.10 릴리즈 노트

## 후반 밸런스 / 엔드리스 압박

- 10분 이후 경험치를 자동 흡수/정산하는 흐름으로 바꿔, 경험치 보석이 투사체를 가리는 상황을 줄였습니다.
- 후반 드랍, 회복, 폭탄, 자석 상호작용을 재조정해 플레이어가 무조건 안전하게 누적 회복/광역 처리를 반복하는 구간을 줄였습니다.
- 엔드리스/합체 보스의 압박을 강화하고, 멀리서 깔짝이는 플레이만으로 끝나는 상황을 줄이도록 보스 이동·탄막·끌어당김·접촉 압박을 보정했습니다.
- 후반 몬스터와 보스 스케일링을 조정해 10분 이후에도 보스 능력과 일반 몬스터 압박이 함께 작동하도록 했습니다.

## 필드 이벤트 / 보상 선택

- 시간 경과 후 깨지는 장판 필드 보상을 즉시 아이템 지급 대신 레벨업과 같은 1장 카드 선택 보상으로 바꿨습니다.
- 필드 보상도 빌드 선택의 일부가 되도록 하여, 후반 아이템 범람과 보상 가시성 문제를 줄였습니다.

## 카드 / 빌드 정보 UI

- 업그레이드 카드에서 현재 수치와 다음 수치를 `전 → 후` 형식으로 보여주고, 변화량을 `+n` 또는 `+%`로 함께 표시합니다.
- 신규 무기 카드도 기본값을 보여주되, 피해량 전체 나열보다 핵심 변화와 기본 성능을 빠르게 읽을 수 있게 정리했습니다.
- 일시정지 화면의 무기/패시브 설명을 현재 수치 중심으로 압축하고, 모달 안에서 좌우 스크롤이 생기지 않도록 정리했습니다.
- 게임 종료/기록 상세에서 탑 피해량, 마지막 피격 원인, 드랍/폭탄/접촉 등 주요 원인을 한글 명사형 라벨로 표시합니다.
- 리더보드 기록을 눌러 해당 기록의 무기 단계, 패시브, 피해량, 처치, 회복, 드랍 통계를 확인할 수 있습니다.

## 무기 이펙트 표시 ON/OFF

- 일시정지 화면에서 무기 아이콘이나 설명 카드를 누르면 해당 무기의 시각 이펙트만 숨기거나 다시 표시할 수 있습니다.
- 이펙트를 숨겨도 무기 발사와 피해는 유지되며, 카드에 `FX ON/OFF` 상태가 표시됩니다.
- 매직 볼트, 레이저, 위성 레이저, 레일건, 번개, 드론, 궤도/오라/노바 계열 렌더링 경로에 무기별 숨김 상태를 반영했습니다.

## 필드 테스트 / 리더보드 안전장치

- `?fieldTest=1` 또는 로컬 환경에서 F9로 필드 테스트 무적 모드를 켤 수 있습니다.
- 한 번이라도 무적 모드를 켠 판은 다시 끄더라도 로컬 최고 기록과 글로벌 리더보드에 제출되지 않습니다.
- 런 스냅샷과 리더보드 API에도 필드 테스트 여부를 반영해 테스트 기록이 공용 기록에 섞이지 않도록 했습니다.
