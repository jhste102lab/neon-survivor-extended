# v1.1.8 릴리즈 노트

## 엔드리스 / 합체 보스 밸런스

- 보스가 포식으로 드랍과 보석을 흡수해도 더 이상 체력을 회복하지 않도록 변경함.
- 보스가 폭탄 드랍을 흡수하면 회복 대신 폭탄 피해가 보스에게 역류하도록 수정함.
- 새 엔드리스 보스가 나올 자리가 없을 때 기존 보스를 강화하는 처리에서 현재 HP 회복을 제거하고, 최대 HP 증가만 남김.
- 합체 보스는 여러 방해 능력을 동시에 갖지 않고 티어별로 1개 능력만 갖도록 정리함.
- 일반/합체 엔드리스 보스 능력 순환에서 드롭 봉인을 제외해 후반에도 드롭 자체는 계속 발생하도록 유지함.
- 합체 보스가 플레이어에게 아주 가까워졌을 때 계속 정면 추격으로 비비지 않고 근접 orbit 이동을 하도록 조정해, 붙는 것은 허용하되 딱 달라붙어 지속 피해를 강제하는 상황을 줄임.

## 포커스 / 대쉬 UI

- TAB 포커스 보관 드랍을 아이템 종류별 3개까지만 붙잡도록 제한함.
- 대쉬 HUD를 화면 중앙 하단의 큰 표시에서 배속 버튼 위의 작은 표시로 축소함.

## 패시브 정리

- 최근 추가했던 지원형 패시브 5종을 제거함.
  - 네온 펄스
  - 위상 방벽
  - 프리즘 표식
  - 중력 앵커
  - 릴레이 코일
- 기존 8개 패시브와 24종 무기 진화 구조는 유지함.

## 참고

- `scripts/smoke-browser.mjs`는 로컬 `playwright` 패키지 부재로 실행하지 못함.
- `scripts/headless-runtime-sim.mjs`의 기능 스모크는 통과했으나, 기존 성능 임계값이 현재 장시간 24x 시뮬레이션과 맞지 않아 프로세스 exit code는 실패로 남음.
- guard 경고는 구조성 advisory만 남았으며 배포 차단 오류는 없음.
