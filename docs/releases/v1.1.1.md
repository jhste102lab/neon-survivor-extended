# v1.1.1 Release Notes

## Highlights

- Fixed global leaderboard submissions for legitimate accelerated runs: the Cloudflare session validator now accepts survival time earned with the in-game 1x/2x/3x speed controls instead of rejecting those scores as `time_ahead_of_session`.
- Added an API regression test that reproduces a legal 3x-style elapsed-time submission before accepting it.
- Added a GitHub source link with an inline GitHub icon on the title screen.
- Rewrote the multilingual README openings to read more like a game introduction, with clearer play-online and “what kind of game is it?” sections.
