# Dimension portal asset attribution

This folder includes selected SVG emoji assets from Google's Noto Emoji project:
https://github.com/googlefonts/noto-emoji

License note from the upstream README: emoji fonts are under SIL Open Font License 1.1; tools and most image resources are under Apache License 2.0. These SVG images are used as decorative dimension portal icons.

Imported files:
- `noto-emoji/emoji_u1f30c.svg` — 탄막 성운
- `noto-emoji/emoji_u26a1.svg` — 기계 감옥
- `noto-emoji/emoji_u1f573.svg` — 중력 우물
- `noto-emoji/emoji_u2696.svg` — 심판자의 결투
- `noto-emoji/emoji_u1f344.svg` — 역병 정원
- `noto-emoji/emoji_u1f537.svg` — 거울 회랑
